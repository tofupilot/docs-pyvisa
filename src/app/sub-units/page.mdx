# Sub-units traceability on TofuPilot

In TofuPilot, units are organized hierarchically. Units within another unit are called **sub-units**. Each sub-unit can belong to only one unit. {{ className: 'lead' }}

<Image src="/nested-units.png" alt="Your profile page" />

## Addind sub-units to an unit

When creating a run on a unit, you can easily specify which units should be included as its sub-units. Simply provide a list of sub-unit serial numbers using the “sub_units” parameter. {{ className: 'lead' }}

    ```python
    client.create_run(
      # other parameters here
      sub-units: [
        {
          "serial-number": "SF050076A4AF00453"
        },
        {
          "serial-number": "SF050076A4AF00454"
        }
      ]
    )
    ```

<Note>
  The “sub-units” parameter adds to the existing sub-units list of the tested
  unit without replacing or removing any original sub-units. Duplicates are
  ignored.
</Note>
