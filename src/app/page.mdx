# TofuPilot Documentation

Welcome to TofuPilot documentation. TofuPilot is a cutting-edge hardware test data management platform that centralizes your test data, creates traceable records, and provides automatic analytics to streamline your testing workflows. {{ className: 'lead' }}

## Value Proposition

TofuPilot allows you to:

- Import your hardware test data from many different sources.
- Automatically create traceable records for each unit.
- Identify bottlenecks and streamline your manufacturing process.
