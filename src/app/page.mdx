# TofuPilot Documentation

Welcome to TofuPilot. TofuPilot is a modern hardware test data management platform that centralizes your test data, ensures precise traceability, and provides automatic analytics to streamline your testing workflows. {{ className: 'lead' }}

## What you can do

TofuPilot allows you to:

- Import your hardware test data from many different sources.
- Automatically create traceable records for each unit.
- Identify bottlenecks and streamline your manufacturing process.

## Getting started

Getting started with TofuPilot is extremely fast. Start by [creating your account](https://www.tofupilot.com/auth/signup).
