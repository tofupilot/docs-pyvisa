# Pytest

Integrate any pytest script with TofuPilot's Pytest Plugin. {{ className: 'lead' }}

<Image src="/pytest-plugin-header.png" alt="" />

## Overview

Many hardware teams choose Python and Pytest for their simplicity and extensive library support. The TofuPilot pytest plugin extends pytest for hardware testing needs.

## Installation

Installing the TofuPilot Python client also installs the pytest plugin and pytest if itâ€™s not already present.

```bash
pip install tofupilot
```

Add it to your test like this:

```python {{ title: 'main.py' }}
import pytest
from tofupilot import conf

conf.set(
    procedure_id="FVT1",
    serial_number="PCB1A001",
    part_number="PCB1",
    revision="A"
)

# Tests for pytest
@numeric_step
def test_one(step):
    step.set_units("V").set_limits(low=3.2, high=3.4)
    value_measured = 3.33
    step.measure(value_measured)
    assert step()

# Rest of your script here

```
Discover more TofuPilot's Pytest Plugin integrations in [Test Runs](/procedures) sections.


### Execution

To run tests using the TofuPilot pytest plugin, use the `--tofupilot` argument.

```bash
pytest --tofupilot main.py
```
